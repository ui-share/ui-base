
🧠 SCSS 구조 설계 전략 요약
분류				전략								설명
1. Abstracts			변수, 믹스인, 함수 등만 위치					디자인 토큰 중심: 변경되더라도 전역 영향
2. Base				글로벌 요소 스타일링						HTML 기본값, 타이포그래피 기본 스타일 지정
3. Layout  			페이지 구조의 큰 틀						Wrapper, Grid, Header, Footer 등 레이아웃 담당
4. Components			UI 컴포넌트 단위로 나누기					재사용 가능한 최소 UI 단위 (버튼, 입력창 등)
5. Pages			특정 페이지 전용 스타일						전역과 충돌하지 않게 분리 적용
6. Themes			다크모드, 브랜드 테마 등 스타일 스위칭			CSS 변수로 테마 구성 추천
7. Vendors			외부 라이브러리 커스터마이징				Bootstrap, Swiper 등 스타일 덮어쓰기
