💡변수 관리 방법

🎨 색상 시스템

예시:
// Gray scale
$gray-100: #F9FAFB;
$gray-200: #F3F5F6;
$gray-300: #DCE0E5;
...
$gray-900: #181A1B;

예시:
@mixin color {
  /* System */
  --color-primary: #{$primary};
  --color-success: #{$success};
  --color-warning: #{$warning};
  --color-error: #{$error};

  /* Text */
  --text-default: #{$gray-900};         // 기본 본문 텍스트
  --text-light: #{$gray-600};           // 서브 텍스트
  --text-disabled: #{$gray-500};        // 비활성 상태
  --text-inverse: #{$white};            // 어두운 배경 대비
  --text-primary: var(--color-primary);
  ...
}
