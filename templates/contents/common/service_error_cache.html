보이는 ARS 화면이 표시되지 않았습니다 보이는 ARS를 안전하게 이용하기 위해 아래 문제를 해결해주세요
1. 중복탭이 열려있는 경우 중복된 탭을 닫고 원본탭을 선택해 주세요. 2. 인터넷 브라우저 창이 많이 열려있을 경우 인터넷 사용 기록(캐시) 삭제 후 재 접속해주세요.
인터넷 사용 기록(캐시) 삭제 방법
크롬 브라우저
  • 크롬 실행 > 상단 점 세게 아이콘 선택
  • 인터넷 사용 기록 삭제 선택
  • 삭제 기간 전체 기간 선택 > 데이터 삭제 > 완료
삼성 브라우저
  • 설정 > 애플리케이션 > 삼성 인터넷 선택
  • 삼성 인터넷 설정 > 인터넷 사용 기록 > 데이터 삭제 > 완료
사파리(Safari) 브라우저
  • 설정 > 앱 > 사파리로 이동
  • 방문 기록 및 웹사이트 데이터 지우기 선택
  • 모든 방문 기록, 모든 탭 닫기선택
  • 방문 기록 지우기 > 완료
